# Stopwatch Web Application

This project is a **Stopwatch Web App** developed using **HTML, CSS, and JavaScript**.
It was created as part of my **Web Development Internship Task (SCT_WD_02)**.

The application allows users to measure time accurately with options to start, pause, reset, and record lap times through an easy-to-use interface.

---

## ✨ Key Features

* ▶️ **Start** the stopwatch to begin timing
* ⏸️ **Pause** the stopwatch anytime
* 🔄 **Reset** the timer back to zero
* 🏁 **Lap Time Tracking** to record intermediate times
* ⏱️ Displays **hours, minutes, seconds, and milliseconds**
* 🎨 Clean and modern **gradient-based UI design**
* 🖱️ Smooth **hover effects and interactive buttons**
* 📱 **Responsive layout** for desktop and mobile devices

## 🎮 How to Use

1. **Start** – Begins the stopwatch timer
2. **Pause** – Temporarily stops the timer
3. **Lap** – Records the current time without stopping the stopwatch
4. **Reset** – Clears the timer and removes all recorded laps
